\section{Repository Structure}

Key files and directories:
\begin{itemize}
\item \code{config.py} --- central configuration (paths, hyperparameters, device defaults)
\item \code{models/gan.py} --- CGAN generator + projection discriminator
\item \code{models/classifier.py} --- FruitCNN classifier
\item \code{train_gan.py} --- CGAN training + FID evaluation + checkpoints/samples
\item \code{train_classifier.py} --- classifier training and evaluation (real/synth/both, data-size control)
\item \code{scripts/generate_synth.py} --- generate synthetic pools from a trained generator checkpoint
\item \code{scripts/run_experiments.py} --- run the full classifier experiment grid
\item \code{scripts/plot_results.py} --- generate classifier plots (accuracy/time/per-class F1)
\item \code{scripts/export_report_tables.py} --- export CSV tables for the report (stdlib-only)
\item \code{scripts/plot_fid_svg.py} --- generate \code{fid_vs_epoch.svg} for the report (stdlib-only)
\item \code{runs/} --- experiment outputs (GAN logs/checkpoints/samples; classifier JSON results/plots)
\item \code{reports/} --- this report and its copied figures/tables for a self-contained hand-in
\end{itemize}
